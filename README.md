# Limit - ハート型人生時計アプリ
自分に残された寿命を１つのハートで表す人生時計アプリです。生年月日・年齢・性別からユーザーに当てはまる平均寿命を割り出し、「平均寿命までに残っている時間」をアプリ内で計算→１つのハート型オブジェクトで表示します。
